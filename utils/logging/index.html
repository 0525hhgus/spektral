init_logging

spektral.utils.init_logging(name=None)

Creates a log directory with an empty log.txt file.

Arguments

  • name: custom name for the log directory (default "%Y-%m-%d-%H-%M-%S")

Return
String, the relative path to the log directory

log

spektral.utils.log(message, print_string=True)

Prints a message to stdout and writes it to the logfile (requires user to call init_logging() at least once in order to save to file).

Arguments

  • message: the string to log;

  • print_string: whether to print the string to stdout;

spektral.utils.model_to_str(model)
