localpooling_filter

spektral.utils.localpooling_filter(adj, symmetric_normalization=True)

Computes the local pooling filter from the given adjacency matrix, as described by Kipf & Welling (2017).

Arguments

  • adj: a np.array or scipy.sparse matrix of rank 2 or 3

  • symmetric_normalization: boolean, whether to normalize the matrix as or as .

Return
The filter matrix, as dense np.array

chebyshev_filter

spektral.utils.chebyshev_filter(adj, k, symmetric_normalization=True)

Computes the Chebyshev filter from the given adjacency matrix, as described in Defferrard et at. (2016).

Arguments

  • adj: a np.array or scipy.sparse matrix

  • k: integer, the order up to which to compute the Chebyshev polynomials

  • symmetric_normalization: boolean, whether to normalize the matrix as or as .

Return
A list of k+1 filter matrices, as np.arrays
